# 主要機能

:::info
このページは作成中です。各項目の詳細は今後追加されます。
:::

## 音響シミュレーション

- **経路追跡ベースのIR生成** — メッシュと材質からインパルス応答を直接合成
- **材質モデル** — absorption、transmission、reflectionを処理
- **リアルタイム更新** — リスナーや音源の移動に応じてIRを更新

## 統合

- **複数プラットフォーム** — macOS、Windows、Linux対応
- **エンジンバインディング** — Unity、Unreal Engine、Web、Pythonを予定

## ツール

- **ExaStudio** — 視覚的ワークフローとデバッグ環境
- **ExaTools** — IR解析、検証、バッチ処理用ツール群
