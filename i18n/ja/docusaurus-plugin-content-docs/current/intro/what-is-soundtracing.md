# Sound Tracingとは?

**Sound Tracing**は、仮想空間内の音源からリスナーへ到達するまでの音響伝搬経路を物理ベースでシミュレーションする技術です。グラフィックスのray tracingに対応する音響領域のアプローチで、反射、回折、吸収などの現象を幾何学的に扱い、空間固有の聴こえ方を再現します。

## 解決すること

- **空間感のある音**: 同じ音源でも洞窟、コンサートホール、路地で異なって聞こえるように、geometryとmaterialを反映したインパルス応答（IR）を生成します。
- **リアルタイム応答**: リスナー、音源、環境が変化したときにIRを更新し、ゲーム、VR、シミュレーターで使える遅延で結果を提供します。
- **物理ベースの一貫性**: ヒューリスティックなリバーブプリセットではなく、geometryから直接導かれた結果を使うため、シーン変化へ自然に適応します。

## 動作の流れ

1. **シーン入力**: メッシュ、材質（absorption/transmission）、音源、リスナー位置を受け取ります。
2. **経路探索**: ray tracingに近い方式で音響伝搬経路を探索し、平面ベースの逆追跡、coplane処理、境界検出など音響向けのアルゴリズムで精度を高めます。
3. **IR合成**: 到達経路の時間、減衰、方向情報を集め、インパルス応答として合成します。
4. **コンボリューション**: IRを音源に畳み込み、最終出力を作ります。
