# Features

:::info
This page is being written. Details for each item will be expanded later.
:::

## Acoustic Simulation

- **Path-tracing-based IR generation** — synthesize impulse responses directly from meshes and materials
- **Material model** — absorption, transmission, and reflection handling
- **Real-time updates** — update IRs in response to listener and source movement

## Integration

- **Multi-platform** — supports macOS, Windows, and Linux
- **Engine bindings** — Unity, Unreal Engine, Web, and Python planned

## Tools

- **ExaStudio** — visual workflow and debugging environment
- **ExaTools** — utilities for IR analysis, validation, and batch processing
