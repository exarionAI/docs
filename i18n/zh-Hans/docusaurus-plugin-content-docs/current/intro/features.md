# 主要功能

:::info
本页正在编写中。各项细节会在后续补充。
:::

## 声学仿真

- **基于路径追踪的 IR 生成** — 直接从 mesh 和 material 合成脉冲响应
- **材质模型** — 处理 absorption、transmission 和 reflection
- **实时更新** — 根据听者和声源移动更新 IR

## 集成

- **多平台** — 支持 macOS、Windows、Linux
- **引擎绑定** — 计划支持 Unity、Unreal Engine、Web、Python

## 工具

- **ExaStudio** — 可视化工作流与调试环境
- **ExaTools** — IR 分析、验证和批处理工具集
