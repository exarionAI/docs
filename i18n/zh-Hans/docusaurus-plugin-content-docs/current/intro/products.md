# 产品线

Sound Tracing 产品线由 **Core 引擎**、**SDK 绑定**、**studio 工具** 和 **分析工具** 四个部分组成。

## Core

基于 C/C++ 的原生声学仿真引擎。

- [STCore](../core/stcore.md) — 第一代 core（包含 FPGA 加速接口，维护中）
- [STCoreV2](../core/stcorev2.md) — 第二代 core（模块化 path module，lock-free）
- [STCore vs STCoreV2 比较](../core/comparison.md)

## SDK

让 Core 引擎可在各平台/语言中使用的绑定阵列。

- [SDK 概览](../sdk/overview.md)
- 计划绑定：Web、Python、Unity、Unreal Engine

## ExaStudio

用于可视化构建和调试声学场景的桌面应用。

- [ExaStudio 概览](../exastudio/overview.md)

## ExaTools

用于 IR 分析、数据转换、批量验证等工作的工具集。

- [ExaTools 概览](../exatools/overview.md)
