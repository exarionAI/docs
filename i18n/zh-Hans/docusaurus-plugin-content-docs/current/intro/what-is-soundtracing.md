# Sound Tracing 是什么？

**Sound Tracing** 是一种基于物理的技术，用于模拟虚拟空间中声源到听者之间的声学传播路径。它对应于图形中的 ray tracing，在声学领域以几何方式处理反射、绕射、吸收等现象，从而重现空间自身的听感。

## 解决什么问题

- **有空间感的声音**：根据 geometry 和 material 等空间信息生成脉冲响应 (IR)，让同一个声源在洞穴、音乐厅、巷道中听起来不同。
- **实时响应**：当听者、声源或环境变化时更新 IR，以适合游戏、VR、模拟器的延迟提供结果。
- **基于物理的一致性**：不依赖启发式混响预设，而是使用直接从 geometry 推导出的结果，因此能自然适应场景变化。

## 如何工作

1. **场景输入**：接收 mesh、material（absorption/transmission）、声源位置和听者位置。
2. **路径追踪**：用类似 ray tracing 的方式搜索声学传播路径，并通过基于平面的反向追踪、coplane 处理、边界检测等声学算法提高精度。
3. **IR 合成**：收集到达路径的时间、衰减和方向信息，合成为脉冲响应。
4. **卷积**：将 IR 与声源信号卷积，生成最终输出。
