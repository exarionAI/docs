# 主要功能

:::info
本頁正在编写中。各項细节會在後續补充。
:::

## 聲學模擬

- **基於路徑追蹤的 IR 產生** — 直接從 mesh 和 material 合成脈衝響應
- **材質模型** — 處理 absorption、transmission 和 reflection
- **即時更新** — 根據聽者和聲源移動更新 IR

## 整合

- **多平台** — 支援 macOS、Windows、Linux
- **引擎綁定** — 計畫支援 Unity、Unreal Engine、Web、Python

## 工具

- **ExaStudio** — 視覺化工作流程與调試環境
- **ExaTools** — IR 分析、驗證和批處理工具集
