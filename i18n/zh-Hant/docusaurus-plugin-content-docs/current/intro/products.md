# 產品線

Sound Tracing 產品線由 **Core 引擎**、**SDK 綁定**、**studio 工具** 和 **分析工具** 四個部分组成。

## Core

基於 C/C++ 的原生聲學模擬引擎。

- [STCore](../core/stcore.md) — 第一代 core（包含 FPGA 加速介面，維護中）
- [STCoreV2](../core/stcorev2.md) — 第二代 core（模組化 path module，lock-free）
- [STCore vs STCoreV2 比較](../core/comparison.md)

## SDK

讓 Core 引擎可在各平台/語言中使用的綁定陣列。

- [SDK 概覽](../sdk/overview.md)
- 計畫綁定：Web、Python、Unity、Unreal Engine

## ExaStudio

用於視覺化建置和调試聲學場景的桌面應用。

- [ExaStudio 概覽](../exastudio/overview.md)

## ExaTools

用於 IR 分析、資料转换、批量驗證等工作的工具集。

- [ExaTools 概覽](../exatools/overview.md)
