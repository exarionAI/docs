# Sound Tracing 是什麼？

**Sound Tracing** 是一種基於物理的技術，用於模擬虛擬空間中聲源到聽者之間的聲學傳播路徑。它對應於圖形中的 ray tracing，在聲學領域以幾何方式處理反射、繞射、吸收等現象，從而重現空間自身的聽感。

## 解決什麼問題

- **有空間感的聲音**：根據 geometry 和 material 等空間資訊產生脈衝響應 (IR)，讓同一個聲源在洞穴、音樂廳、巷道中聽起來不同。
- **即時響應**：當聽者、聲源或環境變化時更新 IR，以適合遊戲、VR、模擬器的延遲提供結果。
- **基於物理的一致性**：不依賴啟發式混響預設，而是使用直接從 geometry 推導出的結果，因此能自然適應場景變化。

## 如何工作

1. **場景輸入**：接收 mesh、material（absorption/transmission）、聲源位置和聽者位置。
2. **路徑追蹤**：用類似 ray tracing 的方式搜尋聲學傳播路徑，並透過基於平面的反向追蹤、coplane 處理、邊界偵測等聲學演算法提升精度。
3. **IR 合成**：收集到達路徑的時間、衰減和方向資訊，合成為脈衝響應。
4. **卷積**：將 IR 與聲源訊號卷積，產生最終輸出。
