# Sound Tracing이란?

**Sound Tracing**은 가상 공간 내 음원이 청취자에게 도달하기까지의 음향 전파 경로를
물리 기반으로 시뮬레이션하는 기술입니다. 빛의 광선 추적(ray tracing)에 대응하는
음향 영역의 접근으로, 반사·회절·흡수 같은 현상을 기하학적으로 다루어 공간이
가지는 고유한 청감을 재현합니다.

## 무엇을 해결하는가

- **공간감 있는 사운드**: 같은 음원이라도 동굴, 콘서트홀, 골목길에서 다르게 들리도록
  공간 정보(지오메트리·재질)를 반영한 임펄스 응답(IR)을 생성합니다.
- **실시간 응답**: 청취자·음원·환경이 변할 때 IR을 갱신하여 게임·VR·시뮬레이터에
  적용 가능한 지연 시간으로 결과를 제공합니다.
- **물리 기반 일관성**: 휴리스틱한 리버브 프리셋 대신 지오메트리에서 직접 유도된
  결과를 사용하여 장면 변화에 자연스럽게 적응합니다.

## 어떻게 동작하는가

1. **장면 입력**: 메시·재질(흡수율·투과율)·음원·청취자 위치를 받습니다.
2. **경로 추적**: 광선 추적과 유사한 방식으로 음향 전파 경로를 탐색하고, 평면 기반
   역추적·동평면(coplane) 처리·경계 검출 등 음향 특성에 맞춘 알고리즘으로 정확도를
   높입니다.
3. **IR 합성**: 도달 경로의 시간·감쇠·방향 정보를 모아 임펄스 응답으로 합성합니다.
4. **컨볼루션**: IR을 음원에 컨볼루션하여 최종 출력을 만듭니다.
