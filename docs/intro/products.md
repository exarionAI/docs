# 제품군

Sound Tracing 제품군은 **Core 엔진**, **SDK 바인딩**, **스튜디오**, **분석 도구**의
네 축으로 구성됩니다.

## Core

C/C++ 기반의 음향 시뮬레이션 네이티브 엔진입니다.

- [STCore](../core/stcore.md) — 1세대 코어 (FPGA 가속 인터페이스 포함, 유지보수)
- [STCoreV2](../core/stcorev2.md) — 2세대 코어 (모듈식 path module, lock-free)
- [STCore vs STCoreV2 비교](../core/comparison.md)

## SDK

Core 엔진을 각 플랫폼/언어에서 사용할 수 있도록 감싸는 바인딩 라인업입니다.

- [SDK 개요](../sdk/overview.md)
- 바인딩(예정): Web, Python, Unity, Unreal Engine

## ExaStudio

음향 장면을 시각적으로 구성하고 디버깅하는 데스크톱 애플리케이션입니다.

- [ExaStudio 개요](../exastudio/overview.md)

## ExaTools

IR 분석, 데이터 변환, 배치 검증 등을 위한 유틸리티 도구 모음입니다.

- [ExaTools 개요](../exatools/overview.md)
