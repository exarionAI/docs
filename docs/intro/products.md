# 제품군

Sound Tracing 제품군은 **코어 SDK**, **스튜디오**, **분석 도구**로 구성됩니다.

## SDK

C/C++ 기반의 음향 시뮬레이션 코어와 각 플랫폼용 바인딩을 제공합니다.

- [SDK 개요](../sdk/overview.md)
- 코어: [STCore](../sdk/core/stcore.md), [STCoreV2](../sdk/core/stcorev2.md)
- 바인딩(예정): Web, Python, Unity, Unreal Engine

## ExaStudio

음향 장면을 시각적으로 구성하고 디버깅하는 데스크톱 애플리케이션입니다.

- [ExaStudio 개요](../exastudio/overview.md)

## ExaTools

IR 분석, 데이터 변환, 배치 검증 등을 위한 유틸리티 도구 모음입니다.

- [ExaTools 개요](../exatools/overview.md)
