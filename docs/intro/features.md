# 주요 기능

:::info
이 페이지는 작성 중입니다. 각 항목의 상세 내용은 추후 보강됩니다.
:::

## 음향 시뮬레이션

- **경로 추적 기반 IR 생성** — 메시·재질로부터 임펄스 응답 직접 합성
- **재질 모델** — 흡수율·투과율·반사 처리
- **실시간 갱신** — 청취자·음원 이동에 반응하는 IR 갱신

## 통합

- **다중 플랫폼** — macOS, Windows, Linux 지원
- **엔진 바인딩** — Unity, Unreal Engine, Web, Python (예정)

## 도구

- **ExaStudio** — 시각적 워크플로우와 디버깅 환경
- **ExaTools** — IR 분석·검증·배치 처리 도구 모음
